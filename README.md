#swipeTo#
swipeTo is a jQuery plugin that emulate the native swipe to action feature upon a list on mobile phones
#Usage#
### DOM
<pre>
  <div class="list">
      <div class="item">
          <a href="#" class="item-swipe">Item</a>
          <div class="item-back">
              ...
          </div>
      </div>
      ...
      <div class="item">
          <a href="#" class="item-swipe">Item</a>
          <div class="item-back">
              ...
          </div>
      </div>
  </div>
</pre>
